# Outreach

## Overview

Supaglue uses the Outreach v2 API.

| Feature                        | Available |
| ------------------------------ | --------- |
| Auth                           | Yes       |
| Managed syncs (common objects) | Yes       |
| Point reads                    | Yes       |
| Action API (common objects)    | Yes       |
| Real-time events               | No        |

Supported common object types:

- Users
- Accounts
- Contacts
- Emails
- Sequences
- Sequence States
- Mailboxes
