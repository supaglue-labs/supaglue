# Marketo

## Overview

Supaglue uses the Marketo REST API.

| Feature                            | Available |
| ---------------------------------- | --------- |
| Authentication                     | Yes       |
| Managed syncs                      | Yes       |
| Actions API                        | No        |
| Real-time events                   | No        |
| Passthrough API                    | Yes       |

Supported common objects:

- Actions
- Automations
- Campaigns
- Contacts
- Emails
- Events
- Lists
- Messages
- Templates
- Users
