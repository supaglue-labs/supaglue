# Salesloft

## Overview

Supaglue uses the Salesloft v2 API.

| Feature                    | Available |
| -------------------------- | --------- |
| Auth                       | Yes       |
| Managed syncs              | Yes       |
| Point reads                | Yes       |
| Action API                 | Yes       |
| Real-time events           | No        |

Supported common objects:

- Users
- Accounts
- Contacts
- Emails
- Sequences
- Sequence States
- Mailboxes
