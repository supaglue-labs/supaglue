# HubSpot

## Overview

Supaglue interfaces with the HubSpot V3 API.

| Feature                    | Available |
| -------------------------- | --------- |
| Auth                       | Yes       |
| Reads: Full refresh        | Yes       |
| Reads: Incremental refresh | Yes       |
| Creates                    | Yes       |
| Updates                    | Yes       |

Supported object types:

- Company
- Contact
- Deal
- User
