# Salesforce

## Overview

Supaglue uses the Salesforce Bulk 2.0 API and the REST API.

| Feature                    | Available |
| -------------------------- | --------- |
| Auth                       | Yes       |
| Reads: Full refresh        | Yes       |
| Reads: Incremental refresh | Yes       |
| Creates                    | Yes       |
| Updates                    | Yes       |
| Handles rate limits        | Yes       |

Supported object types:

- Account
- Contact
- Lead
- Opportunity
- User
