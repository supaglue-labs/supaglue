# Pipedrive

## Overview

Supaglue interfaces with the Pipedrive V1 API.

| Feature                            | Available |
| ---------------------------------- | --------- |
| Authentication                     | Yes       |
| Managed syncs                      | Yes       |
| Actions API                        | Yes       |
| Real-time events                   | No        |
| Passthrough API                    | Yes       |

Supported common objects:

- Account
- Contact
- Lead
- Opportunity
- User
