# Marketo

## Overview

Supaglue uses the Marketo REST API.

| Feature                    | Available |
| -------------------------- | --------- |
| Auth                       | Yes       |
| Reads: Full refresh        | Yes       |
| Reads: Incremental refresh | Yes       |
| Creates                    | Yes       |
| Updates                    | Yes       |

Supported object types:

- Actions
- Automations
- Campaigns
- Contacts
- Emails
- Events
- Lists
- Messages
- Templates
- Users
