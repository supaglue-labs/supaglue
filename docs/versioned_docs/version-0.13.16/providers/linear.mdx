# Linear

## Overview

| Feature                 | Available |
| ----------------------- | --------- |
| Authentication (Oauth2) | Yes       |
| Managed syncs           | Yes       |
| Actions API             | No        |
| Real-time events        | No        |
| Passthrough API         | Yes       |

Supported Standard Objects: `issues`, `comments`, `teams`, `projects`, `users`.

Scopes: `read`, `write`, `issues:create`, `comments:create`. (Alternatively, set `admin` if you want full access to everything.)
