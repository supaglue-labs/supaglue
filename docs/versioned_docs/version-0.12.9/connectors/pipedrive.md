# Pipedrive

## Overview

Supaglue interfaces with the Pipedrive V1 API.

| Feature                        | Available |
| ------------------------------ | --------- |
| Auth                           | Yes       |
| Managed syncs (common objects) | Yes       |
| Point reads                    | Yes       |
| Action API (common objects)    | Yes       |
| Real-time events               | No        |

Supported common objects:

- Account
- Contact
- Lead
- Opportunity
- User
