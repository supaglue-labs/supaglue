# Outreach

## Overview

Supaglue uses the Outreach v2 API.

| Feature                    | Available |
| -------------------------- | --------- |
| Auth                       | Yes       |
| Managed syncs              | Yes       |
| Point reads                | Yes       |
| Creates                    | Yes       |
| Updates                    | Yes       |
| Real-time events           | No        |

Supported object types:

- Users
- Accounts
- Contacts
- Emails
- Sequences
- Sequence States
- Mailboxes
