# Pipedrive

## Overview

**Status:** beta

**Category:** `crm`

Supaglue uses the Pipedrive V1 API.

| Feature                              | Available                                       |
| ------------------------------------ | ----------------------------------------------- |
| Authentication (`oauth2`)            | Yes                                             |
| Managed syncs                        | Yes                                             |
| &nbsp;&nbsp;&nbsp; Sync strategies   | `full then incremental` (soft delete supported) |
| Unified API                          | Yes                                             |
| &nbsp;&nbsp;&nbsp; Data invalidation | Yes                                             |
| Real-time events                     | No                                              |
| Passthrough API                      | Yes                                             |

#### Supported common objects:

- Account
- Contact
- Lead
- Opportunity
- User
