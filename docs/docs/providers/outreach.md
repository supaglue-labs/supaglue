# Outreach

## Overview

Supaglue uses the Outreach v2 API.

| Feature                            | Available |
| ---------------------------------- | --------- |
| Authentication                     | Yes       |
| Managed syncs                      | Yes       |
| Actions API                        | Yes       |
| Real-time events                   | No        |
| Passthrough API                    | Yes       |

Supported common objects:

- Users
- Contacts
- Sequences
- Sequence States
- Mailboxes
