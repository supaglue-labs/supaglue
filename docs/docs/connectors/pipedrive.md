# Pipedrive

## Overview

Supaglue interfaces with the Pipedrive V1 API.

| Feature                    | Available |
| -------------------------- | --------- |
| Auth                       | Yes       |
| Managed syncs              | Yes       |
| Point reads                | Yes       |
| Creates                    | Yes       |
| Updates                    | Yes       |
| Real-time events           | No        |

Supported object types:

- Account
- Contact
- Lead
- Opportunity
- User
