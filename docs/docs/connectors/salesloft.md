# Salesloft

## Overview

Supaglue uses the Salesloft v2 API.

| Feature                    | Available |
| -------------------------- | --------- |
| Auth                       | Yes       |
| Reads: Full refresh        | Yes       |
| Reads: Incremental refresh | Yes       |
| Creates                    | Yes       |
| Updates                    | Yes       |

Supported object types:

- Users
- Accounts
- Contacts
- Emails
- Sequences
- Mailboxes
