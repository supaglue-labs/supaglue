# Zoho CRM

## Overview

Supaglue interfaces with the Zoho REST V3 API.

| Feature                    | Available |
| -------------------------- | --------- |
| Auth                       | Yes       |
| Reads: Full refresh        | No        |
| Reads: Incremental refresh | No        |
| Creates                    | No        |
| Updates                    | No        |
| Handles rate limits        | No        |
