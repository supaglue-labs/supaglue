# Gong

## Overview

Supaglue interfaces with the Gong V2 API.

| Feature                          | Available |
| -------------------------------- | --------- |
| Auth                             | Yes       |
| Managed syncs (standard objects) | Yes       |
| Point reads                      | No        |
| Action API                       | No        |
| Real-time events                 | No        |

Supported standard objects: `call`, `detailedCall`, `callTranscript`.
