# Marketo

## Overview

Supaglue uses the Marketo REST API.

| Feature                    | Available |
| -------------------------- | --------- |
| Auth                       | Yes       |
| Managed syncs              | Yes       |
| Point reads                | Yes       |
| Action API                 | Yes       |
| Real-time events           | No        |

Supported common objects:

- Actions
- Automations
- Campaigns
- Contacts
- Emails
- Events
- Lists
- Messages
- Templates
- Users
